# DOM-шаблоны

Календарь предоставляет удобную возможность настраивать HTML-разметку с помощью параметра `DOMTemplates`. Это позволяет добавить собственные элементы, такие как кнопки или любой другой HTML-элемент, в календарь.

`DOMTemplates` принимает `type` календаря в качестве ключа и строку в качестве значения.

В следующем примере заголовок календаря кастомизирован для `type: 'default'`, и внутри календаря добавляется обычная кнопка.
